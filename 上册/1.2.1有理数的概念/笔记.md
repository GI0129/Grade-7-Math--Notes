# 笔记
**思考****:**

2 = 2/1,  3 = 3/1,  7 = 7/1.  **正整数 → 可以写成 → 正分数**

\-3 = -3/1, -5 = -5/1, -11 = -11/1.  **负整数 → 可以写成 → 负分数**

0 = 0/1,  **0 → 可以写成 → 分数**

**整数 → 可以写成 → 分数**

0.2 = 1/5,  -1.23 = -123/100.  **有限小数 → 可以写成 → 分数**

0.333… = ⅓,  0.111… = 1/9.  **无限循环小鼠 → 可以写成 → 分数**

> [!NOTE]
> **有理数的概念:**  可以写成**分数**形式的数称为**有理数**
